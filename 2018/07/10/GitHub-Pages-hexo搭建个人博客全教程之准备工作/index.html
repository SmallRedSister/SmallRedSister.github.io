GitHub Pages+hexo搭建个人博客全教程之准备工作

撰写于 2018-07-10 修改于 2018-07-10 分类 搭建个人博客 标签 GitHub Pages+hexo Git Node.js Sublime Text Typora MPic IPic Bmob 域名

看到别人有自己的个人技术博客，自己也想要一个，于是走上了搭建（zhuang）个人(bi)博客之路。起初我是想要一个网页版的简历（我会放到个人博客网站），我是使用GitHub Pages+hexo实现的，也推荐大家使用该方法（主要是服务器免费）。本人半科班出道，安卓开发，有些前端基础，如果你也想拥有自己的博客网站，我想接下来的文章会让你少走弯路。

你需要的准备工作：

目录 是否必备 为什么要用它？
GitHub 是 我们用到GitHub Pages做为服务器，必须有一个GitHub账号
Git 是 git用来分布式管理项目（博客就是项目）
Node.js 是 为了让网站有更好的效果
Sublime Text 可选 这个东西可以帮助我们打开各种格式的文件并修改其内容
Typora 可选 后期写博客神器，推荐
MPic 可选 图床神器，和Typora配合使用，如果你是Mac系统，使用IPic
Bmob 可选 这个还是挺好的，后期存图片，博客加载图片的来源
域名 可选 我是使用自己域名的，这东西自己考虑一下，推荐你有一个

域名

我们先聊域名，如何拥用自己的域名，推荐去阿里云购买一个（买最便宜的），登录阿里云（没有号注册），

全部导航—>产品—>域名与网站—>点击域名注册
