写在前面

​ 网站搭建起来了，也装饰好，一切妥当，只欠东风（充实内容啊）；单有躯壳是不行的，还需要灵魂。没说错的话，估计我又开始废话了。这篇我们主要解决两个问题，1.使用什么工具写博客，2.博客中插入图片问题，

Hexo 编写博客工具

​ 工具推荐使用Typora，不知道Typora或者还没安装Typora请看我的上一篇博客，程序员必备神器（Typora）。你也可以使用Sublime Text 3 来写，Sublime Text 3 写的话需要写MD语法。我是两者一起配合使用😁。

Hexo 在博文中插入图片

  • 插入本地图片
  • 插入有链接的图片

插入本地图片

插入本地图片需要修改主题_config.yml配置文件post_asset_folder项为true。

今后你new 一篇博客时/source/_posts文件夹内除了xxxx.md文件还有一个同名的文件夹，把图片放到里面即可.

最后在xxxx.md中想引入图片时，先把图片复制到xxxx这个文件夹中，然后只需要在xxxx.md中按照markdown的格式引入图片：![你想输入的替代文字](xxxx/图片名.jpg)

插入有链接的图片

​ 考虑到本地存放图片图片，一占用电脑内存，二来博客项目过于膨大，想一下，一篇博客三四张图片，10篇20篇以后呢?所以我们不能只有一种对策方法，凡事都有三种对策方法才行。当然了，本地图片也有好处，存放更换频繁的、重要的、隐私的图片最佳选择。使用第三方平台保存图片的话，推荐使用Bmob和七牛云+MPic，我只讲Bmob。Bmob对于个人有应用来说再好不过，一个月有20G免费存储空间。

Bmob先注册一个好号，然后创建一个应用，最后上传素材获取图片链接。功能很强大，用一次就知道了哈，
